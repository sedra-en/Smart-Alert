# Smart Alert — Customer Sentiment Analysis System

A real-time facial emotion detection system built to enhance 
customer satisfaction in service centers.

## Problem
Traditional feedback methods (surveys, interviews) are slow and 
inaccurate. Smart Alert detects customer emotions in real time — 
no surveys needed.

## Features
- Real-time face detection & emotion classification
  (Happy, Sad, Angry, Fear, Surprise, Disgust, Neutral)
- Smart alerts to staff when negative emotions are detected
- Analytics dashboard for management
- Supports multi-face detection in a single frame
- Privacy-first: no personal data stored without consent

## Tech Stack
Python · CNN / Deep Learning · OpenCV · Flask  
HTML · CSS · JavaScript · MySQL

## Model Performance
- Target accuracy: 85%+
- Processing speed: < 1 second per frame
- Trained on real-world image datasets (Kaggle)
